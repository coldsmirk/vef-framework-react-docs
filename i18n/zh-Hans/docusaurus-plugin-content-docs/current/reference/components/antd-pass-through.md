# Antd 透传组件

这一页覆盖 `components` 包中**主要只是透传 Antd 官方组件**的导出。  
这些组件的文档策略统一如下:

- 推荐场景简要说明
- Props 说明直接指向 Antd 官方
- 不重复维护一整份镜像文档
